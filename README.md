# Integração do GPT 3 com Whatsapp

Permite interação através de mensagens de texto pelo WhatsApp com AI GPT3.
### Setup

1 - npm install
<br>
2 - npm start
<br>
3 - (no whatsapp) comandos: /bot 'o que quer '; /img 'o que quer'
